# Lira - Premium E-Commerce Platform

## 🚀 Features

- ✅ OTP Email Verification via Brevo
- ✅ MongoDB Database
- ✅ 5-Leg Binary Tree
- ✅ 10 Level Income Distribution
- ✅ Franchise System with Pincode
- ✅ 12 Month Wallet
- ✅ Pool System (Franchise & General)
- ✅ Admin Panel
- ✅ Beautiful Email Notifications
- ✅ Mobile Responsive Design
- ✅ Birthday & Anniversary Greetings
- ✅ Best Performers Dashboard

## 🛠️ Tech Stack

- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Email**: Brevo (Sendinblue)
- **Frontend**: HTML5, CSS3, JavaScript
- **Deployment**: Render
